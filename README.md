# focus-forge
FocusForge is your ultimate productivity companion, designed to help you stay focused, organized, and efficient. This single-page web application combines essential productivity tools in one sleek, dark-themed interface with vibrant electric blue accents, animations, and interactive elements.

Features

Responsive Dark-Themed Design
FocusForge features a modern dark UI with electric blue highlights, providing an aesthetic that is easy on the eyes, reduces strain during long work sessions, and looks professional on both desktop and mobile devices.

Navbar & Theme Toggle

Sticky navigation bar for easy access to all sections.

Theme toggle button allows switching between light and dark modes seamlessly.

To-Do List

Add tasks quickly with a simple input field.

Mark tasks as completed by clicking on them; completed tasks are visually crossed out and tracked.

Delete tasks with a one-click button.

Automatic progress bar that updates as you complete tasks, giving a visual sense of accomplishment.

Pomodoro Timer

Classic 25-minute Pomodoro timer with start, pause, and reset functionality.

Session count tracker to monitor the number of completed Pomodoro sessions.

Alerts when a Pomodoro session is completed, keeping you on track.

Progress Tracking

Real-time progress bar for your completed tasks.

Count of completed tasks updates automatically as you manage your to-do list.

Interactive Animations & Hover Effects

Smooth hover effects on buttons and menu cards to enhance user experience.

Subtle animations when elements appear on the page, making the interface more dynamic and engaging.

Fully Responsive Layout

Optimized for desktops, tablets, and mobile devices.

Grid layouts automatically adjust to different screen sizes for a consistent user experience.

AI Assistance

AI tools were used to enhance the design by selecting color schemes, improving readability, and optimizing layout structures.

All AI assistance was used responsibly to support design and coding decisions while ensuring custom functionality.

Technology Stack

HTML5 – for structuring content.

CSS3 – for styling, dark mode, animations, hover effects, and responsive design.

JavaScript (Vanilla JS) – for interactive functionality:

To-Do List logic

Pomodoro Timer

Theme Toggle

Progress tracking

No external libraries were used, keeping the project lightweight and fast.
